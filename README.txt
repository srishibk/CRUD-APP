# CRUD-APP
This is an simple CRUD APP using node.js,angular.js and Mongo DB

REQUIREMENTS:
1.NODE.JS and MONGO DB.
2.requires node server and mongo server running.
3.create a public folder and save index.html . Create controllers folder inside public folder and save controller.js file.
4.create contactlist database IN MONGODB.
5.install node modules.
6.url is localhost:3000.
